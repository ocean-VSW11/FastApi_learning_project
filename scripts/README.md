# 工具脚本说明

这个目录包含了项目的各种工具脚本和实用程序。

## 脚本列表

### init_db.py
**用途**: 初始化数据库
**描述**: 创建数据库表结构，为项目首次运行做准备。

**使用方法**:
```bash
python scripts/init_db.py
```

**功能**:
- 创建所有数据库表
- 设置数据库连接
- 初始化基础数据结构

### create_test_data.py
**用途**: 创建测试数据
**描述**: 在数据库中创建示例用户、文章和分类数据，用于开发和测试。

**使用方法**:
```bash
python scripts/create_test_data.py
```

**创建的数据**:
- 3个测试用户（admin, user1, user2）
- 示例文章数据
- 基础分类数据

### demo_api.py
**用途**: API 演示和测试
**描述**: 演示如何使用项目的各个 API 端点，包括认证、CRUD 操作等。

**使用方法**:
```bash
python scripts/demo_api.py
```

**演示功能**:
- 用户登录认证
- 文章的增删改查
- 分类管理
- API 调用示例

## 使用建议

### 首次设置项目
1. 运行 `init_db.py` 初始化数据库
2. 运行 `create_test_data.py` 创建测试数据
3. 启动 FastAPI 服务器
4. 运行 `demo_api.py` 验证 API 功能

### 开发过程中
- 使用 `demo_api.py` 测试新开发的 API 功能
- 需要重置数据时，重新运行 `create_test_data.py`

## 注意事项

- 运行脚本前确保已安装项目依赖
- 确保数据库文件有适当的读写权限
- 在生产环境中谨慎使用这些脚本，特别是数据创建脚本
